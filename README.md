## About Me

Hello there ,
Im **Kuril**, a **Full-Stack Web Developer** based in Jakarta, Indonesia, specializing in the **PERN Stack**  
(PostgreSQL 路 Express 路 React 路 Node.js).

I build **production-ready web applications** with a strong focus on **frontend architecture and long-term maintainability**, including :

- **Clean Architecture (Frontend & Backend)**
- **Feature-Based & Domain-Oriented Frontend Structure**
- **Domain-Driven Design (DDD) on the Backend**
- **Repository Pattern & SOLID Principles**

Im particularly interested in **frontend engineering**, focusing on scalable React applications, predictable state management, and clean separation between UI, domain, and data layers.

**Currently expanding :** Next.js 路 NestJS 路 CI/CD 路 Advanced system design  
**Open to roles :** Full-Stack Developer 路 Frontend Developer  
**Work preference :** Remote or Hybrid
