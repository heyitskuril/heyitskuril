##  About Me

Im **Kuril**, a **Full-Stack Web Developer** based in Jakarta, Indonesia, specializing in the **PERN Stack**  
(PostgreSQL 路 Express 路 React 路 Node.js).

I build **production-ready web applications** with a strong focus on:

- **Domain-Driven Design (DDD)**
- **Clean & Layered Architecture**
- **Repository Pattern**
- **SOLID Principles**
- **Long-term maintainability & scalability**

My work emphasizes **clear separation of concerns**, predictable systems, and codebases that are easy to evolve.

**Currently expanding:** Next.js 路 NestJS 路 CI/CD 路 Advanced system design  
**Open to roles:** Full-Stack Developer 路 Frontend Developer  
**Work preference:** Remote or Hybrid
